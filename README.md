# Educational Keystroke Logger (Python + Tkinter)

This project demonstrates how keystrokes can be captured within an application
for educational cybersecurity purposes.

## Features
- CLI + GUI based execution
- Logs keystrokes typed inside the application window
- User-controlled start/stop logging
- Timestamped log files

## Disclaimer
This tool is intended strictly for educational and authorized testing.
It does NOT capture system-wide keystrokes or operate in the background.
